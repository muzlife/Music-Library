운영 홈 활용 매뉴얼
  1. 상품 보유 여부나 현재 위치를 빠르게 확인해야 할 때 이 화면을 엽니다.
  2. 곡명, 아티스트명, 상품명, 바코드, 견출지 ID로 먼저 상품을 조회합니다.
  3. 현재 위치와 직전 위치를 확인해 장식장 동선을 판단합니다.
  4. 수정이나 재배치가 필요하면 관리자만 상단 관리로 이동합니다.
운영 홈은 조회 전용입니다. 상품 수정, 이동, 등록/삭제는 관리에서만 처리합니다.

대시보드 활용 매뉴얼

  2. 장식장을 클릭한 뒤 열/칸을 선택해 현재 배치를 확인합니다.
  3. 선반 / 썸네일 / 리스트 중 목적에 맞는 보기 방식을 고르고, 상품을 선택해 이동·정렬·복구를 처리합니다.
  4. 우측 이동 작업대에서 미배치 상품이나 검색 결과를 체크한 뒤, 대상 칸을 클릭해 배치를 마무리합니다.
