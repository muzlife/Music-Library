Critical 위반 — 즉시 수정

2건
🚫
THE LILA BAN — 퍼플 액센트 전면 사용
--selected-accent: #8b5cf6, --selected-accent-strong: #7c3aed, --size-accent-oversize: #7c3aed 등 보라계열이 선택 상태 핵심 컬러로 전체에 걸쳐 사용 중. taste-skill은 "AI Purple/Blue 미학은 전면 금지(BANNED)"로 명시하며, 네온 글로우나 보라 버튼 등을 엄격히 차단합니다. 브랜드 컬러 Teal(#0f766e)이 이미 있으므로 선택 상태도 이 계열로 통일해야 합니다.
taste-skill §3 Rule 2 · "THE LILA BAN: The AI Purple/Blue aesthetic is strictly BANNED."
🚫
액센트 컬러 10종 — "Max 1 Accent" 규칙 위반
현재 size별 컬러가 8종(amber, blue, teal, teal-light, cyan, purple, pink, indigo)이고 선택 상태용 purple, 브랜드 teal, 액센트 brown까지 포함하면 11개 이상의 액센트 컬러가 혼재합니다. taste-skill은 "Max 1 Accent Color, Saturation < 80%"를 요구합니다. 포맷 식별 컬러는 기능적 구분으로 유지하되, 채도를 낮추고 공통 기준을 세워야 합니다.
taste-skill §3 Rule 2 · "Constraint: Max 1 Accent Color. Saturation < 80%."

Warning 개선사항

4건
⚠️
숫자에 모노스페이스 폰트 미적용
VISUAL_DENSITY 기준 이 UI는 7~8 범주(Cockpit Mode)입니다. taste-skill은 "VISUAL_DENSITY > 7에서 모든 숫자는 Monospace(font-mono)를 사용"으로 명시합니다. 현재 대시보드의 통계값, 슬롯 번호, 위치 코드 등 숫자가 본문 폰트(Pretendard)로 표시되어 있어 정렬이 흔들리고 밀도감이 떨어집니다. JetBrains Mono 또는 Fira Code를 숫자 전용으로 페어링하세요.
taste-skill §6 VISUAL_DENSITY 8-10 · "Mandatory: Use Monospace (font-mono) for all numbers."
⚠️
카드 과다 사용 — border / divide 패턴으로 전환 필요
대시보드 패널들이 모두 card + shadow 컨테이너로 묶여 있어 경계가 중첩됩니다. taste-skill §3 Rule 4는 VISUAL_DENSITY > 7 환경에서는 "카드 컨테이너 BANNED, border-t / divide-y / 여백으로만 구분"을 요구합니다. 특히 대시보드 통계 영역, 운영 홈 미니카드 등은 카드 없이 순수 구분선+여백으로도 충분히 계층을 표현할 수 있습니다.
taste-skill §3 Rule 4 · "DASHBOARD HARDENING: For VISUAL_DENSITY > 7, generic card containers are strictly BANNED."
⚠️
야간 모드 주황 액센트 과포화
--theme-shell-accent: #ff7a1a (Night Mode)는 명도와 채도가 높아 어두운 배경에서 과하게 튑니다. taste-skill은 "Saturation < 80%"와 "Desaturate accents to blend elegantly with neutrals"를 요구합니다. #e8620a 정도로 채도를 낮추고, 포커스/호버 상태에만 강하게 표현하는 전략이 적합합니다.
taste-skill §3 Rule 2 · §7 Visual & CSS · "NO Oversaturated Accents: Desaturate accents to blend elegantly with neutrals."
⚠️
3-칼럼 동일 카드 레이아웃 사용 패턴
대시보드 영웅 카드, 운영 홈 미니카드 영역에서 3개 동일 컬럼 패턴이 반복됩니다. taste-skill은 이를 "THE GENERIC 3-EQUAL-CARD LAYOUT IS BANNED"으로 정의하며, 2-column Zig-Zag, 비대칭 그리드, 또는 수평 스크롤 방식을 강제합니다. 어드민 콘솔 특성상 완전한 제거는 어렵지만, 주요 Hero 영역만이라도 비대칭(2+1 또는 1+wide) 구조로 전환을 권장합니다.
taste-skill §7 Layout · "NO 3-Column Card Layouts: The generic 3 equal cards horizontally feature row is BANNED."

통과 — 잘 지켜진 항목

7건
  • ✅ No Pure Black: --ink: #1f2a37 — Off-Black 사용 준수. #000000 없음.
  • ✅ Diffuse Shadow: 0 16px 36px rgba(15,23,42,0.07) — 넓게 퍼지는 미묘한 그림자 사용. 네온 아우터 글로우 없음.
  • ✅ CSS Custom Properties 시스템: 모든 색상, 반경, 쉐도우가 변수로 체계화되어 테마 전환이 완전히 작동.
  • ✅ 다크/라이트 테마 구현: data-theme="night" / "day" 완전히 구현. 배경에 ambient gradient 사용.
  • ✅ 한국어 폰트 최적화: Pretendard + Apple SD Gothic Neo + Noto Sans KR 올바른 fallback 체인 구성.
  • ✅ Liquid Glass 내부 굴절 보더: inset 0 1px 0 rgba(255,255,255,0.4) 내부 하이라이트 다수 사용. taste-skill §4 "Liquid Glass Refraction" 준수.
  • ✅ Empty/Error 상태: 페이지 도움말, 메뉴얼 블록, 오류 메시지 패턴이 구현되어 있음. taste-skill §3 Rule 5 준수.

/* taste-skill 기준 적용 | 2026-05 */

:root {
  /* ── Primary accent (Teal only — Max 1 Accent) ── */
  --brand:           #0f766e;       /* PRIMARY — 모든 선택/액션에 사용 */
  --brand-active:    #14b8a6;       /* hover / focus 상태 */
  --brand-soft:      #d9f4f0;       /* 선택 배경, chip 배경 */
  --brand-ring:      rgba(15,118,110,0.28); /* focus ring */
  --brand-shadow:    rgba(15,118,110,0.14); /* 선택 shadow */

  /* ── REMOVED: Purple accent (LILA BAN) ── */
  --selected-accent: #8b5cf6;  ← 삭제
  --selected-accent-strong: #7c3aed;  ← 삭제
  --selected-surface: #f3e8ff;  ← 삭제
  --selected-ring: rgba(139,92,246,0.36);  ← 삭제

  /* ── Selected state → Teal 계열로 교체 ── */
  --selected-accent:        #0f766e;
  --selected-accent-strong: #0d6660;
  --selected-surface:       #d9f4f0;
  --selected-surface-soft:  #f0faf9;
  --selected-ring:          rgba(15,118,110,0.28);
  --selected-shadow-soft:   rgba(15,118,110,0.12);

  /* ── Format type chips — 채도 낮춤 (Saturation < 65%) ── */
  --size-accent-std:       #c47b30;   /* 기존 #b45309 → 명도+, 채도- */
  --size-accent-book:      #5d89b4;   /* 기존 #2563eb → 탈포화 */
  --size-accent-lp:        #4a9e98;   /* 기존 #0f766e → 살짝 밝게 */
  --size-accent-lp10:      #5eaaa4;   /* 기존 #14b8a6 → 탈포화 */
  --size-accent-lp7:       #4f8fa8;   /* 기존 #0891b2 → 탈포화 */
  --size-accent-oversize:  #7a7aa8;   /* 기존 #7c3aed → 완전 탈포화 */
  --size-accent-cassette:  #b06b8b;   /* 기존 #db2777 → 탈포화 */
  --size-accent-tape:      #6670a8;   /* 기존 #4338ca → 탈포화 */

  /* ── Typography — 숫자 전용 mono 추가 ── */
  --font-sans:   "Pretendard", "Apple SD Gothic Neo", "Noto Sans KR", system-ui;
  --font-mono:   "JetBrains Mono", "Fira Code", "Cascadia Code", monospace;
  /* 적용 → 모든 숫자, 슬롯 코드, 통계값에: font-family: var(--font-mono) */

  /* ── Night mode accent — 채도 낮춤 ── */
  /* 기존 --theme-shell-accent: #ff7a1a → */
  --theme-shell-accent:    #e8620a;   /* Saturation 72% (기존 100%) */
}
