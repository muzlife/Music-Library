# Left Nav Submenu + Header Slim Layout Design

## Context
운영/관리 콘솔 좌측 네비게이션이 1차 탭만 표시되고, 서브 메뉴가 노출되지 않아 이동성이 떨어진다. 또한 상단 헤더(히어로 영역)가 과도하게 커져 관리 화면의 밀도를 해친다. 카메라 기능은 상단 유틸에 있어 접근성이 낮고, ERD/매뉴얼 링크도 좌측 네비게이션 하단으로 이동하는 것이 자연스럽다.

## Goals
- 좌측 네비게이션에 서브 메뉴를 노출해 이동 경로를 명확히 한다.
- 상단 헤더는 언어 선택과 운영/관리 전환만 남기고 슬림화한다.
- 카메라 메뉴를 좌측 네비게이션에 고정 배치한다.
- ERD 요약/상세/툴 활용 매뉴얼 링크를 좌측 네비게이션 하단에 배치한다.

## Non-Goals
- 라우팅 구조 변경 또는 API 변경
- 기존 탭/패널의 구조적 리팩터링
- UI 테마/색상 대대적 변경

## UX Overview
### Left Nav (관리 모드 전용)
- 좌측 네비게이션은 **관리 모드(admin shell)**에서만 표시한다.
- 1차 탭 라벨은 기존과 동일하게 **대시보드 / 미디어 / 컬렉터블 / 운영/연계 / 카메라**로 유지한다.
- 활성 1차 탭 아래에 해당 서브 메뉴를 표시한다.
- ERD 요약/ERD 상세/툴 활용 매뉴얼은 좌측 메뉴 하단 “Docs” 섹션으로 이동한다.

#### Submenu Mapping
- **대시보드**: 서브메뉴 없음 (서브메뉴 영역 숨김)
- **미디어**: 검색 / 관리 / 등록 / 소스
- **컬렉터블**: 검색 / 관리 / 등록
- **운영/연계**: 장식장 / 슬롯 / 예외 큐 / 계정 / 연동/API 설정 / 백업/내보내기 / 메타 동기화
- **카메라**: 서브메뉴 없음 (서브메뉴 영역 숨김)

#### Left Nav Ordering
- 상단부터: 대시보드 → 미디어 → 컬렉터블 → 운영/연계 → 카메라
- 하단 고정 섹션: Docs (ERD 요약 / ERD 상세 / 툴 활용 매뉴얼)
- **운영 모드에서는 Docs 섹션을 숨긴다.**

### Header
- 히어로 텍스트/아트는 숨긴다.
- 헤더에는 언어 선택 + 운영(hahahoho)/관리 전환 버튼만 표시한다.
- 헤더 높이를 낮춰 상단 공간을 최소화한다.
  - 관리 모드: 언어 선택 + 운영으로 전환 버튼
  - 운영 모드: 언어 선택 + 관리로 전환 버튼 + 카메라 버튼

## Behavior Details
- 서브 메뉴 클릭 시 기존 탭/서브탭 전환 로직을 사용한다.
- 활성 상태는 현재 탭/서브탭과 동기화한다.
- 서브메뉴가 없는 1차 탭에서는 서브메뉴 영역을 숨긴다.
- 모바일 레이아웃에서는 기존 Drawer 동작을 유지한다.
- 모바일 Drawer에서는 **활성 1차 탭의 서브메뉴만 표시**한다.
- 기존 탭 내부 서브탭 UI는 **유지**하되, 좌측 서브메뉴가 주 이동 경로가 되도록 시각적 우선순위를 준다.
- 카메라 메뉴는 **관리 모드 좌측 네비게이션**으로 이동한다.
- 운영 모드에서는 카메라 접근을 **상단 유틸 버튼**으로 유지한다.
- 운영/연계 탭의 서브탭 라인(`tabOps` 내부 `subtabs`)에 있는 카메라 버튼은 **중복이므로 제거**한다.

## Implementation Notes
- `app/static/index.html` 내 좌측 네비게이션에 서브 메뉴 컨테이너 추가
- 서브 메뉴 활성/표시 상태는 기존 `mediaMode`, `goodsMode`, `switchSubTab` 상태를 기준으로 계산
- **관리 모드** 상단 유틸에서 `tabCameraBtn` 제거 후 좌측 네비게이션으로 이동
- `shellUtilityBar`의 ERD/매뉴얼 링크를 좌측 네비게이션 하단으로 이동
- 헤더는 `admin-shell-hero--nav-only`, `ops-home-hero--nav-only` 스타일로 슬림화

## Success Criteria
- 좌측 네비게이션에 서브 메뉴가 표시되고 클릭 시 올바른 탭으로 이동한다.
- 헤더는 언어/운영·관리 버튼만 보이고 높이가 축소된다.
- 관리 모드에서는 카메라 메뉴가 좌측 네비게이션에 고정된다.
- 운영 모드에서는 카메라 버튼이 상단 유틸에 유지된다.
- ERD 요약/상세/매뉴얼 링크가 좌측 네비게이션 하단에 표시된다.

## Test Checklist
- 데스크톱: 서브 메뉴 표시/활성 상태 확인
- 데스크톱: 헤더 높이 축소 및 언어/전환 버튼만 보이는지 확인
- 모바일: 좌측 Drawer에서 동일 메뉴 표시되는지 확인
- 카메라 메뉴 좌측 고정 표시 여부 확인
- ERD/매뉴얼 링크 정상 동작 확인
