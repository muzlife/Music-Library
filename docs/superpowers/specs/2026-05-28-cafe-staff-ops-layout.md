## 구성 요소별 상세

### 1. 헤더
- 제목 "☕ 카페 운영"
- 총 요청 건수 (실시간 갱신)
- WebSocket 연결 상태 (🟢/🟡)

### 2. 칸반 보드 (3컬럼)

| 컬럼 | 내용 | 카드 액션 |
|------|------|----------|
| **대기** | REQUESTED 상태 | `[재생]` `[보류]` |
| **처리중** | PLAYING 상태 | `[완료]` |
| **완료** | RETURNED/CANCELLED | 없음 (최근 15건) |

**카드 정보:**
- 🎵 Spotify / 💿 로컬 구분 아이콘
- 곡명 (최대 60자)
- 아티스트
- 테이블 번호 (주황색 배지)

### 3. 재생 바
- 현재 재생 중인 곡명 + 아티스트
- `[⏸ 일시정지]` `[⏹ 정지]`
- Spotify/로컬 공통 제어

### 4. 반응 피드 (WebSocket 실시간)
- 시간 + 테이블번호 + 이모지 + 내용
- 최신 항목이 위로 쌓임
- 최대 120px 높이, 스크롤

## 확장 예정 영역

```
┌──────────────────────────────────────────────────────────────────┐
│  [🔍 로컬 검색] [📋 대기열] [📊 통계] [⚙️ 설정]                     │  서브탭
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│  (선택된 서브탭 콘텐츠)                                            │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

| 서브탭 | 기능 |
|--------|------|
| **로컬 검색** | NAS 파일 검색 → 즉시 재생 |
| **대기열** | 연속 재생 목록 관리 (드래그 순서 변경) |
| **통계** | 오늘 요청 수, 인기곡, 시간대별 분포 |
| **설정** | Spotify Connect 상태, 오디오 출력, 태블릿 관리 |
